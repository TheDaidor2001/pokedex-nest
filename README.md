# Ejecutar desarrollo

1. Clonar el repositorio
2. Ejecutar el comando 
```
yarn install
```
3. Tener nest CLI instalado
```
npm i -g @nestjs/cli
```

4. Levantar la base de datos mediante Docker
```
docker compose up -d
```
